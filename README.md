# Calculadora em C

Projeto desenvolvido em linguagem C com o objetivo de praticar lógica de programação.

## Funcionalidades
- Soma
- Subtração
- Multiplicação
- Divisão

## Tecnologias
- Linguagem C

## Como executar

Compile:
gcc calculadora.c -o calculadora

Execute:
./calculadora
